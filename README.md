## 🦇 Overview

**Gotham File Vault** is a dark, atmospheric file management application inspired by the dark streets of Gotham City. Built with **Python** and **Streamlit**, it provides a sleek, noir-themed interface for managing your digital files with Batman-style flair.

Create, Read, Update, and Delete files through an immersive dark UI featuring glowing effects, custom CSS styling, and a truly villainous user experience that even the Joker would appreciate.

---

## ✨ Features

### 🌟 Core Functionality
- ✅ **Create Files** - Add new files to your vault with dark elegance
- ✅ **Read Files** - Browse and view your stored secrets with a single click
- ✅ **Update Files** - Modify existing files in the shadows
- ✅ **Delete Files** - Remove files permanently from the vault

## 🚀 Quick Start

### Prerequisites
- Python 3.8 or higher
- pip (Python package installer)

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/CodeXRudy/gotham-file-vault.git
   cd gotham-file-vault
   ```

2. **Create a Virtual Environment** (Optional but Recommended)
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

### Running the Application

```bash
streamlit run app.py
```

The application will open in your default browser at `http://localhost:8501`

## 🎮 How to Use

### Creating a File
1. Navigate to the **Create** section
2. Enter your filename and content
3. Click **Save to Vault** - your file is now secure in Gotham

### Reading a File
1. View the **Vault Contents** section
2. Select a file from the list
3. Read your file in the display area

### Updating a File
1. Open the **Update** section
2. Select the file you wish to modify
3. Edit the content and save changes
4. Your file is updated in the shadows

### Deleting a File
1. Go to the **Delete** section
2. Choose the file to remove
3. Confirm deletion - it vanishes into the darkness

## 🎨 Customization

The application uses a dark Batman-inspired theme. You can customize colors, effects, and styling by modifying `assets/style.css`:

```css
/* Dark background - pure black like Gotham nights */
background-color: #0a0a0a;

/* Glowing accent colors */
--batman-yellow: #fdb913;
--gotham-purple: #4a0080;
--shadow-glow: 0 0 20px rgba(253, 185, 19, 0.3);
```

## 🔐 Security Notes

- Files are stored locally in the application directory
- No data is transmitted to external servers
- For production use, consider:
  - Implementing file encryption
  - Adding user authentication
  - Using secure file storage solutions
  - Regular backups of vault contents
